DATA LANDSCAPES – SPARKLINES

Hand-drawn data visualizations by artist Doug MacDowell

Description: Data Landscapes are a series of seven hand-drafted data visualizations, each representing six data lines spanning 2000–2024. These analog charts reference Artist/Hacker-made coffee-maker computers from 2002, 2018, 2019, and 2024. The drawings analyze a 15-year gap in these creations using publicly available social data.

Each visualization is drawn by hand using pre-computer drafting tools: t-square, triangle, rulers, lettering kit, and templates. Rather than the 20 minutes a computer would take, each drawing requires 20–30 hours. Part of the art exhibition Sparklines, these and other hand-drafted data visualizations accompany a coffee maker computer.
